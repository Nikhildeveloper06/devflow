A lightweight, real-time Kanban project management tool — built as a full-stack case study project.

## Status
🚧 In active development

## Stack
- Frontend: React (Vite)
- Backend: Node.js + Express
- Database: PostgreSQL
- Real-time: Socket.io
- Auth: JWT

## Structure
- `/client` — React frontend
- `/server` — Express backend + API
